Configuration Mode

Select how you would like to report spam (straight to the registrar, to Spam Cop, or to both).

Default Action

Select what to do after reporting the email.

WHOIS URL Selection

Select the default WHOIS webpage to load up during a failed lookup with the Just Report It API.

Custom Abuse Email Lookup Server

Set a custom abuse email lookup server instead of using the official Just Report It API.

Example: https://api.justreport.it/lookup/
Example: 123456789

Spam Cop Configuration

Specify your SpamCop ID to enable sending reports to Spam Cop.
Your SpamCop ID can be found in the provided report email address: submit.{SpamCopID}@spam.spamcop.net

Example: bJ2asdjlka

Custom Report Address

Specify one or multiple custom email addresses to send reports to.

Example: abuse@email1.com,abuse@email2.com
